Cloudinary expressions let you use dynamic values — such as the asset's width, face count, or a user-defined variable — as transformation parameters. They are composed using the `CLDExpression` class, which supports arithmetic operations and serialises to Cloudinary's expression syntax.

`CLDConditionExpression` extends `CLDExpression` with comparison and logical operators, enabling conditional transformations with `ifCondition` / `ifElse` / `endIf`.

`CLDVariable` lets you define named variables and reference them inside expressions.

---

### Predefined variable factory methods

Each method returns a `CLDExpression` seeded with the corresponding predefined variable. You can then chain arithmetic operators.

| Method | URL abbreviation | Description |
|---|---|---|
| `CLDExpression.width()` | `w` | Current asset width in pixels. |
| `CLDExpression.height()` | `h` | Current asset height in pixels. |
| `CLDExpression.initialWidth()` | `iw` | Original uploaded asset width. |
| `CLDExpression.initialHeight()` | `ih` | Original uploaded asset height. |
| `CLDExpression.aspectRatio()` | `ar` | Current asset aspect ratio (width / height). |
| `CLDExpression.initialAspectRatio()` | `iar` | Original uploaded asset aspect ratio. |
| `CLDExpression.pageCount()` | `pc` | Number of pages (PDF, animated GIF). |
| `CLDExpression.faceCount()` | `fc` | Number of faces detected. |
| `CLDExpression.illustrationScore()` | `ils` | Illustration score from AI content analysis (0–1). |
| `CLDExpression.currentPageIndex()` | `cp` | Index of the current page (multi-page assets). |
| `CLDExpression.pageXOffset()` | `px` | Page X offset. |
| `CLDExpression.pageYOffset()` | `py` | Page Y offset. |
| `CLDExpression.tags()` | `tags` | Asset tags. |
| `CLDExpression.duration()` | `du` | Video duration in seconds. |
| `CLDExpression.initialDuration()` | `idu` | Original video duration in seconds. |

### Serialisation

---

### Examples

```swift
// Set width to half the original width
let expr = CLDExpression.initialWidth().divide(by: 2)
CLDTransformation().setWidth(expr)
// URL: w_iw_div_2

// Set height to maintain aspect ratio relative to a fixed width
let height = CLDExpression.initialHeight()
    .multiple(by: CLDExpression.width())
    .divide(by: CLDExpression.initialWidth())
CLDTransformation()
    .setWidth(400)
    .setHeight(height)
// URL: w_400,h_ih_mul_w_div_iw
```

### then()

Finalise the condition and attach it to the related `CLDTransformation` as an `if` parameter.

```swift
public func then() -> CLDTransformation
```

---

### Examples

```swift
// Fluent condition using the instance method ifCondition()
CLDTransformation()
    .ifCondition()
        .width(">", 1000)
    .then()
        .setWidth(800)
        .setCrop(.scale)
    .endIf()
// URL: if_w_gt_1000/w_800,c_scale/if_end

// Condition with and
CLDTransformation()
    .ifCondition(
        CLDConditionExpression.faceCount()
            .greater(then: 0)
            .and()
            .width(">", 300)
    )
    .setGravity(.faces)
    .setCrop(.thumb)
    .setWidth(200)
    .endIf()
// URL: if_fc_gt_0_and_w_gt_300/g_faces,c_thumb,w_200/if_end

// Condition + else branch
CLDTransformation()
    .ifCondition()
        .aspectRatio(">", 1.0)
    .then()
        .setWidth(800)
        .setCrop(.scale)
    .chain()
    .ifElse()
        .setHeight(600)
        .setCrop(.scale)
    .endIf()
```

## CLDVariable

```swift
@objcMembers open class CLDVariable: NSObject
```

`CLDVariable` defines a user-defined variable that you can reference inside transformation expressions.

### Properties

| Property | Type | Description |
|---|---|---|
| `name` | `String` | Variable name including the `$` prefix (added automatically if omitted). |
| `value` | `String` | The serialised variable value. |

### Initializers

```swift
public init(name: String, value: String)
public init(name: String, value: Double)
public init(name: String, value: Int)
public init(name: String, values: [String])
```

<ParamField path="name" type="String">
  Variable name. Must match `$[a-zA-Z][a-zA-Z0-9]*`. The `$` prefix is prepended automatically if missing.
</ParamField>

<ParamField path="value" type="String | Double | Int">
  The variable value. Numeric types are converted to their string representations.
</ParamField>

<ParamField path="values" type="[String]">
  An array of strings, serialised as a Cloudinary list literal using `!value1:value2!` syntax.
</ParamField>

<Note>
  A variable is only valid when its name matches the pattern `$[a-zA-Z][a-zA-Z0-9]*` and its value is non-empty. Invalid variables are silently ignored when passed to `setVariable(_:)`.
</Note>

### Methods

```swift
public func asString() -> String
public func asParams() -> [String: String]
```

`asString()` returns `"$name_value"`. `asParams()` returns `["$name": "value"]`.

---

### Setting variables on a transformation

```swift
// Using CLDVariable
let widthVar = CLDVariable(name: "myWidth", value: 400)
CLDTransformation()
    .setVariable(widthVar)
    .setWidth("$myWidth")
    .setCrop(.scale)

// Using convenience overloads directly on CLDTransformation
CLDTransformation()
    .setVariable("myWidth", int: 400)
    .setWidth("$myWidth")

// Setting multiple variables at once
let vars = [
    CLDVariable(name: "w", value: 400),
    CLDVariable(name: "h", value: 300),
]
CLDTransformation()
    .setVariables(vars)
    .setWidth("$w")
    .setHeight("$h")
    .setCrop(.fill)

// Using a list variable
let listVar = CLDVariable(name: "colors", values: ["red", "blue", "green"])
CLDTransformation().setVariable(listVar)
// Produces: $colors_!red:blue:green!
```
