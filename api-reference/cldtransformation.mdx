---
title: 'CLDTransformation'
description: 'Build image and video transformations for Cloudinary delivery URLs using the CLDTransformation class.'
keywords: ['cloudinary', 'ios', 'transformation', 'CLDTransformation', 'crop', 'resize', 'effect']
---

`CLDTransformation` represents a full set of transformations applied on-the-fly by Cloudinary to an image or video asset. Each setter method returns `Self`, so you can chain calls in a fluent style.

```swift
let transformation = CLDTransformation()
    .setWidth(800)
    .setHeight(600)
    .setCrop(.fill)
    .setGravity(.face)
    .setQuality(.auto())
```

When passed to a URL builder, the transformation is serialised as a slash-separated string of comma-separated parameter pairs:

```
w_800,h_600,c_fill,g_face,q_auto
```

<Accordion title="Cropping">

### setCrop

Set the crop mode. Use the `CLDCrop` enum or a raw string.

```swift
open func setCrop(_ crop: CLDCrop) -> Self
open func setCrop(_ crop: String) -> Self
```

<ParamField path="crop" type="CLDCrop | String">
  The crop mode to apply.
</ParamField>

#### CLDCrop values

| Case | URL value | Description |
|---|---|---|
| `.fill` | `fill` | Resize and crop to the exact requested dimensions, keeping as much of the asset as possible. |
| `.crop` | `crop` | Extract a region of the given width and height from the original asset. |
| `.scale` | `scale` | Scale the asset to the given dimensions. Both dimensions are scaled. |
| `.fit` | `fit` | Scale the asset to fit entirely within the bounding box, preserving aspect ratio. |
| `.limit` | `limit` | Like `fit` but only if the original is larger than the requested dimensions. |
| `.mFit` | `mfit` | Like `fit` but only downscales. |
| `.lFill` | `lfill` | Like `fill` but only if the original is larger than the requested dimensions. |
| `.pad` | `pad` | Scale and pad to the exact dimensions. |
| `.lPad` | `lpad` | Like `pad` but only if the original is larger. |
| `.mPad` | `mpad` | Like `pad` but only downscales. |
| `.thumb` | `thumb` | Smart thumbnail generation using face detection. |
| `.imaggaCrop` | `imagga_crop` | Crop using the Imagga add-on. |
| `.imaggaScale` | `imagga_scale` | Scale using the Imagga add-on. |

```swift
CLDTransformation()
    .setWidth(300)
    .setHeight(300)
    .setCrop(.fill)
    .setGravity(.face)
```

### setQuality

Control the compression quality of the output asset.

```swift
open func setQuality(_ quality: CLDQuality) -> Self
open func setQuality(_ quality: String) -> Self
```

<ParamField path="quality" type="CLDQuality | String">
  A `CLDQuality` configuration object or a raw quality string.
</ParamField>

Use the `CLDQuality` factory methods:

| Method | URL value | Description |
|---|---|---|
| `CLDQuality.fixed(_ level: Int)` | `q_<level>` | Fixed quality level from 1 (lowest) to 100 (highest). |
| `CLDQuality.auto()` | `q_auto` | Cloudinary selects the optimal quality automatically. |
| `CLDQuality.auto(.best)` | `q_auto:best` | Least aggressive automatic optimisation. |
| `CLDQuality.auto(.good)` | `q_auto:good` | Balanced automatic optimisation. |
| `CLDQuality.auto(.eco)` | `q_auto:eco` | More aggressive automatic optimisation. |
| `CLDQuality.auto(.low)` | `q_auto:low` | Most aggressive automatic optimisation (smallest files). |
| `CLDQuality.jpegMini()` | `q_jpegmini` | Uses the JPEGmini add-on. |

```swift
CLDTransformation().setQuality(.auto(.good))
CLDTransformation().setQuality(.fixed(85))
```

<Note>
  `setQuality(_ quality: Int)` is deprecated. Use `CLDQuality.fixed(_:)` instead.
</Note>

## chain()

Call `chain()` to finalize the current set of transformation parameters and start a new transformation step. Each step becomes a `/`-separated segment in the final URL.

```swift
@discardableResult
open func chain() -> Self
```

```swift
let transformation = CLDTransformation()
    .setWidth(800)
    .setHeight(600)
    .setCrop(.fill)
    .chain()           // → first URL segment: w_800,h_600,c_fill
    .setEffect(.sepia) // → second URL segment: e_sepia

// Produces: w_800,h_600,c_fill/e_sepia
print(transformation.asString() ?? "")
```

## CLDEagerTransformation

`CLDEagerTransformation` extends `CLDTransformation` and adds a `setFormat` method for specifying the output format when creating eager transformations at upload time.

```swift
open class CLDEagerTransformation: CLDTransformation
```

### setFormat

```swift
open func setFormat(_ format: String?) -> Self
```

<ParamField path="format" type="String?">
  The output format extension, e.g. `"jpg"`, `"webp"`, `"mp4"`. Appended as a `/format` suffix after the transformation string.
</ParamField>

```swift
let eager = CLDEagerTransformation()
    .setWidth(800)
    .setCrop(.fill)
    .setFormat("webp")
```
