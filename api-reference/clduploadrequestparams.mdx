---
title: 'CLDUploadRequestParams'
description: 'Configure every aspect of a Cloudinary upload request—public ID, transformations, metadata, analysis, and more.'
---

`CLDUploadRequestParams` holds the parameters sent alongside an upload request. Pass an instance to any `CLDUploader` upload method via the `params` argument. All setter methods return `Self`, so you can chain them.

```swift
let params = CLDUploadRequestParams()
    .setPublicId("profile-photo")
    .setFolder("users/avatars")
    .setTags(["profile", "avatar"])
    .setTransformation(CLDTransformation().setWidth(800).setCrop("limit"))
```

<Note>
  `CLDUploadRequestParams` inherits `setResourceType(_:)`, `setSignature(_:)`, and `setApiKey(_:)` from its base class `CLDRequestParams`. These are documented separately below.
</Note>

## Identification

### setPublicId(_:)

```swift
@discardableResult
open func setPublicId(_ publicId: String) -> Self
```

Sets the identifier used to access the uploaded resource. A randomly generated ID is assigned if you do not call this method. The public ID may include a folder path using `/` as a separator.

<ParamField path="publicId" type="String" required>
  The public ID to assign. May contain forward slashes to place the resource in a folder (e.g., `"users/123/avatar"`).
</ParamField>

```swift
params.setPublicId("products/shoe-red-42")
```

## Filename handling

### setUseFilename(_:)

```swift
@discardableResult
open func setUseFilename(_ useFilename: Bool) -> Self
```

When `true`, the original filename is used as the public ID (normalized, with random characters appended for uniqueness). Default: `false`.

<ParamField path="useFilename" type="Bool" required>
  Pass `true` to derive the public ID from the filename.
</ParamField>

---

### setUniqueFilename(_:)

```swift
@discardableResult
open func setUniqueFilename(_ uniqueFilename: Bool) -> Self
```

When `false`, no random characters are appended to the filename-derived public ID. Only relevant when `setUseFilename(true)` is also set. Default: `true`.

<ParamField path="uniqueFilename" type="Bool" required>
  Pass `false` to use the exact filename without uniqueness suffix.
</ParamField>

---

### setDiscardOriginalFilename(_:)

```swift
@discardableResult
open func setDiscardOriginalFilename(_ discardOriginalFilename: Bool) -> Self
```

When `true`, the original filename is discarded when the resource is delivered as an attachment. Default: `false`.

<ParamField path="discardOriginalFilename" type="Bool" required>
  Pass `true` to discard the filename in attachment deliveries.
</ParamField>

---

### setFilenameOverride(_:)

```swift
@discardableResult
open func setFilenameOverride(_ filenameOverride: String) -> Self
```

Overrides the original asset name with the given value.

<ParamField path="filenameOverride" type="String" required>
  The new filename to use instead of the original.
