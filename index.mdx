## Get started in 3 steps

<Steps>
  <Step title="Install the SDK">
    Add Cloudinary to your project using your preferred dependency manager.

    ```ruby Podfile
    pod 'Cloudinary', '~> 5.0'
    ```
  </Step>
  <Step title="Configure your credentials">
    Create a `CLDConfiguration` with your Cloudinary cloud name and API key, then initialize `CLDCloudinary`.

    ```swift
    let config = CLDConfiguration(cloudName: "YOUR_CLOUD_NAME", apiKey: "YOUR_API_KEY")
    let cloudinary = CLDCloudinary(configuration: config)
    ```
  </Step>
  <Step title="Upload and display">
    Upload a file and generate a transformed URL for display.

    ```swift
    // Upload a file
    cloudinary.createUploader().upload(file: fileUrl, params: nil)

    // Generate a 300x300 fill crop URL
    let url = cloudinary.createUrl()
        .setTransformation(CLDTransformation().setWidth(300).setHeight(300).setCrop(.fill))
        .generate("sample.jpg")
    ```
  </Step>
</Steps>
